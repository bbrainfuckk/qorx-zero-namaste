# Public/private IP boundary

This repository is intentionally a clean-room Hackathon Edition.

## Included

- Standard browser IndexedDB persistence
- A small, documented ranking formula
- TTL and explicit record deletion
- A capped text proof frame
- Thin OpenAI and Qwen Responses API adapters
- UI, tests, notebook, and submission documentation

## Excluded

- Private Qorx compiler or language implementation
- Qorx Local Pro source or binaries
- Private routing, verification, compression, or benchmark systems
- Internal datasets, prompts, credentials, build history, or private docs
- Any dependency on an unpublished Qorx component

The public application remains complete without those systems. The AGPL license
protects this edition's terms, but no public source license can prevent people
from studying public code. The security control is therefore exclusion, not
obfuscation.
